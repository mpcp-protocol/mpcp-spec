# Contributing to MPCP Specification

Thank you for your interest in contributing to the Machine Payment Control Protocol (MPCP) specification.

## Overview

This repository (`mpcp-spec`) contains the **canonical protocol specification**. The reference implementation lives in [mpcp-reference](https://github.com/mpcp-protocol/mpcp-reference).

## What to Contribute

- **Protocol changes** — Artifact definitions, verification rules, canonicalization, hashing
- **Architecture docs** — System model, actors, flows
- **Clarifications** — Ambiguities, examples, normative text
- **Roadmap** — Implementation and evolution planning

## Process

1. Open an issue to discuss proposed changes
2. Create a branch and submit a pull request
3. Changes to normative protocol text require review and consensus
4. See [VERSIONING.md](VERSIONING.md) for compatibility policy

## Structure

```
mpcp-spec/
├── docs/
│   ├── overview/      # Problem, goals, what is MPCP
│   ├── architecture/  # System model, actors, flows
│   ├── protocol/      # Artifact specs, verification, hashing
│   └── guides/        # Conceptual guides
├── diagrams/
├── roadmap/
└── ...
```

## Questions
